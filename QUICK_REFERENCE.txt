================================================================================
📋 配置说明
================================================================================

1. api_key
   - 说明: 火山引擎 API 密钥
   - 获取: 火山引擎控制台 → API Key 管理
   - 示例: 65e7792c-3f9a-4dd3-9467-da68bad8ebc3

2. Model
   - 说明: 要使用的大模型名称
   - 轻量级: doubao-seed-2-0-mini-260215 ⭐ 推荐
   - 高性能: doubao-pro-4-turbo

3. api_url
   - 说明: 火山引擎 API 端点
   - 北京区: https://ark.cn-beijing.volces.com/api/v3 ⭐ 推荐
   - 上海区: https://ark.cn-shanghai.volces.com/api/v3
   - 香港区: https://ark.ap-hongkong.volces.com/api/v3
