🚀 快速开始

✅ 项目已构建完成！所有文件都已创建，可以立即使用。

安装步骤

  1. 安装依赖
    pip install -r requirements.txt
  2. 配置 API Key（可选）

    编辑 .env 文件，添加你的火山引擎 API Key

    VOLC_API_KEY=your_api_key_here
  3. 启动服务
    python main.py
  4. 访问文档

    在浏览器中打开：http://localhost:8000/docs

❓ 常见问题

Q: 没有 API Key 能使用吗？

可以！大模型相关功能会被禁用，但数据处理和模型训练仍正常工作。

Q: 模型预测时列不匹配怎么办？

已自动处理！代码会自动对齐特征列到训练时的列。

Q: PCA 降维何时触发？

当特征数 > 2 时，K-Means 聚类会自动使用 PCA 降维到 2D 用于可视化。

Q: 如何修改服务端口？

在 main.py 最后修改 uvicorn.run 中的 port 参数。
